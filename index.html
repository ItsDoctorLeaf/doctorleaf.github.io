MineCraft Plugins

The Pressure

Don't let it get to you...

The pressure is one of my most recent game projects. It's a game where you are in a submarine sinking to the bottom of the ocean! You realise that the primary reinforcement of your ship is buckling under pressure. Your only chance is to repair it constantly. You use a variety of tools to repair damage and cracks. This game was made in a few days for a gameJam challenge.

Poison Control

Stay toxic.

Poison control was another game that I made for a Jam in which I had 72 hours. It was a game themed upon poison, as such I wanted to make a rage game. Confusing puzzles, Painful controls, A long map with big risks. Checkpoints are available along the way if you complete small puzzles.

Plants VS Zombies : Pollution

A remake with a twist.

This game was made when I got aseprite. I was trying to test my artistic abilities, so I made this game! It is not playable as I like not being sued atm, and don't want to risk it, however, you can find a play-through on my YouTube! In this game, the plants die after different amounts of time, this means that you have to have a constant replacement for each plant ready. There are a few different types of plants that are more useful in different scenarios. For example, the pea shooter is good for single enemies whilst the lightning reed is good for massive hoards. Each plant has different amounts of health, so they will all die at different times. This is an infinite game with no set ending.



MineCraft Plugins

Character Creations

A Roleplay Character Creator for MineCraft Java

This is a java plugin where you create your own Roleplay characters. It was made to be used for a server, which never ended up being created. In this plugin, you can create your own custom roleplay characters with names and genders. You can search for people's real name and roleplay names and can ban certain phrases from being used as RP names.

Chaos Plugin

A Chaotic Plugin for MineCraft Java 1.20

The chaos plugin is inspired by the GTA V chaos mod. I wanted to create a similar sort of chaotic event, whilst making it easy to use. The chaos plugin doesn't use mods, and instead runs server side, making it easy to implement and work well for multiplayer. I created over 40 custom events that range from spawning a couple of enemies, summoning lighting around the player, and explosions... lots of explosions.
